# Entrega-final-del-proyecto

## Descripción del Proyecto

Freelancer es una aplicación web desarrollada como proyecto académico de Ingeniería de Software. Su objetivo es proporcionar una plataforma digital donde profesionales independientes, diseñadores y desarrolladores puedan presentar sus servicios, habilidades y proyectos de forma organizada y atractiva.

La aplicación permite visualizar información profesional, servicios ofrecidos y un formulario de contacto, facilitando la comunicación entre potenciales clientes y el profesional.

---

## Objetivo General

Desarrollar un portafolio web que permita mostrar información profesional y servicios de un freelancer mediante una interfaz intuitiva y responsive.

---

## Características

* Página principal de presentación profesional.
* Sección de servicios ofrecidos.
* Formulario de contacto.
* Página de inicio de sesión.
* Página administrativa para futuras funcionalidades.
* Diseño responsive adaptable a diferentes dispositivos.
* Navegación sencilla e intuitiva.
